19      ''' 
20      Apply a butterworth filter to the data. 
21   
22      fp is the passband frequency in Hz 
23      fs is the stopband frequency in Hz 
24   
25      Example of Notations: 
26      --------------------- 
27      For fs = 10, fp = 15 a high-pass filter is applied with the transition band between 10-15 Hz 
28      For fs = 15, fp = 10 a low-pass filter is applied with the transition band between 10-15 Hz 
29      For fs = [10, 40], fp = [20,30] a bandpass-filter with the passband 20-30 Hz and transitions 
30                                      between 10-20 Hz and 30-40 Hz is applied 
31      For fs = [20, 30], fp = [10,40] a bandstop-filter with the stopband 20-30 Hz and transitions 
32                                      between 10-20 Hz and 30-40 Hz is applied 
33   
34      Input: 
35      ------ 
36      -- data - numpy array 
37      -- fp - float - pass-band frequencies 
38      -- fs - float - stop-band frequencies 
39      -- s_rate - float - the sampling rate in Hz 
40      -- gpass - float - maximal attenuation in the passband (in dB) 
41      -- gstop - float - minimal attenuation in the passband (in dB) 
42      -- axis - int -along which axis the filter is applied 
43      -- zero_phase - bool - if zero phase filter is applied by filtering in both directions 
44      -- return_param - bool - if the order, b, and a shoud be returned 
45   
46      Output: 
47      ------- 
48      -- data - numpy array - the filtered data array 
49      if return_param: 
50         -- data 
51         -- ord - int - filter order (this must be doubled for the zero_phase implementation) 
52         -- (b, a) - Numerator (`b`) and denominator (`a`) polynomials of the IIR filter. 
53      ''' 
54      # get filter parameters 
55      wp, ws, ftype = _preparefilter(fp, fs, s_rate) 
56      if zero_phase == True: 
57          gpass = gpass / 2. # half for filtering in both directions 
58          gstop = gstop / 2. 
59          filt = _signal.filtfilt 
60      else: 
61          filt = _signal.lfilter 
62      ord, wn = _signal.buttord(wp=wp, ws=ws, gpass=gpass, gstop=gstop) 
63      b,a = _signal.butter(N=ord, Wn = wn, btype=ftype) 
64      data = filt(b=b, a=a, x=data, axis=axis) 
65      if return_param: 
66          return data, ord, (b, a) 
67      return data
