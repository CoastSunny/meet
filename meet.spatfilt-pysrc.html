 19      """ 
 20      Common Spatial Pattern (CSP) 
 21   
 22      This takes the multivariate data in two conditions and finds the spatial filters which minimize 
 23      the variance in condition 2, while simultaneously maximizing the variance in condition one. 
 24   
 25      The algorithm uses Singular Value Decomposition since this is more stable than the Eigenvalue 
 26      Decomposition of the Covariance Matrices 
 27   
 28      The filters are scaled such, that Var(cond1) + Var(cond2) = 1. The eigenvalues give the variance 
 29      in condition 1. 
 30      The filters are in the columns of the filter matrix. Patterns can be obtained by inverting the filter matrix 
 31      and are located in the rows of the inverted filter matrix. 
 32   
 33      filter, eigval = CSP(data1, data2) 
 34      filtered_data = filter.T.dot(data1) 
 35   
 36      Input: 
 37      ------ 
 38      -- data1 - 2d array - data in condition 1 (variables in rows, observations in columns) 
 39      -- data2 - 2d array - data in condition 2 (variables in rows, observations in columns) 
 40         The number of variables in data1 and data2 must be equal 
 41      -- center - bool - if data should be centered, defaults to True 
 42   
 43      Output: 
 44      ------- 
 45      -- filter - where the individual filters are in the columns of the matrix (in order of decreasing eigenvalues) 
 46      -- eigvals - eigenvalues in decreasing order 
 47      """ 
 48      p1, n1 = data1.shape 
 49      p2, n2 = data2.shape 
 50      #remove mean 
 51      if center: 
 52          data1 = _signal.detrend(data1, axis=1, type='constant') 
 53          data2 = _signal.detrend(data2, axis=1, type='constant') 
 54      #normalize by signal length to have equal weights on all points 
 55      data1 = data1 
 56      data2 = data2 / (n2/float(n1)) 
 57      #whiten for data2 
 58      W1, s = _linalg.svd(data2, full_matrices = False)[:2] 
 59      #get rank 
 60      rank1 = (s > (_np.max(s) * _np.max(data2.shape) * _np.finfo(data2.dtype).eps)).sum() 
 61      W = W1[:,:rank1] / s[:rank1][_np.newaxis] # scale by singular values to get whitening matrix 
 62      W2, s2 = _linalg.svd(W.T.dot(data1), full_matrices = False)[:2] 
 63      rank2 = (s2 > (_np.max(s2) * _np.max([W.shape[1], data1.shape[1]]) * _np.finfo(data1.dtype).eps)).sum() 
 64      both = W.dot(W2[:,:rank2]) 
 65      # scale by eigenvalue 
 66      s2 = (s2[:rank2]**2) / (s2[:rank2]**2 +1) 
 67      both = both * _np.sqrt((1-s2))[_np.newaxis] * _np.sqrt(n1) 
 68      return both, s2
